## 🎯 О проекте
**actitracker** — это бесплатная альтернатива подобным приложениям, уже доступным на маркетах. Такие приложения созданы для тех, кто хочет отслеживать, чем он занимается в течение дня, и для кого важно найти структуру своего дня, а, возможно, и всей жизни. Если это приложение поможет вам в этом, я буду очень рад!

## 🚀 Быстрый старт
1. Создайте активности
2. Сгруппируйте их по тегам
3. Добавьте некоторые активности в панель уведомлений для доступа из любого места, включая экран блокировки
4. Включайте и выключайте таймеры активностей
5. В конце дня, недели, месяца или года проанализируйте время, затраченное на активности и их категории, на специальной круговой диаграмме

## ✨ Ключевые особенности
- Отслеживание времени по активностям
- Группировка через теги
- Отчеты по дням / неделям / месяцам
- Быстрый доступ через панель уведомлений
- Настройка внешнего вида
