### 🚀 1. Aktivitätsliste erstellen
Um eine Aktivität zu erstellen, klicken Sie auf die Schaltfläche „+“

| unten auf dem „Heute“-Bildschirm | oder | oben rechts auf dem „Verwalten“-Bildschirm |
| :---: | :---: | :---: |
|<img src="../../screenshots/scr_home_add_button.png" height="300"> | | <img src="../../screenshots/scr_manage_activities_add_button.png" height="300"> |

Im angezeigten Fenster <img src="../../screenshots/scr_home_add_dialog.png" height="200"> können Sie Folgendes konfigurieren:

| Farbe der Aktivität | Icon für die Aktivität | Tag zuweisen<br>(siehe Punkt 2) |
|:---: | :---: | :---: |
|<img src="../../screenshots/scr_home_add_colorpicker.png" height="450"> | <img src="../../screenshots/scr_home_add_iconpicker.png" height="450"> | <img src="../../screenshots/scr_home_add_tagpicker.png" height="450"> |

Danach können Sie Aktivitäten durch Antippen auf dem „Heute“-Bildschirm sowie in der Benachrichtigungsleiste im entsperrten und gesperrten Zustand des Geräts starten und stoppen (siehe Punkt 4 zur Einrichtung des Schnellzugriffs). Die Aktivitäten sehen wie folgt aus:

| Inaktiv | Aktive Zeiterfassung |
|:---: | :---: |
|<img src="../../screenshots/scr_home_inactive.png" height="450"> | <img src="../../screenshots/scr_home_active.png" height="450"> |

Im rechten Bild sehen Sie, dass auf dem „Heute“-Bildschirm die aktive Aktivität auch in einer speziellen Zeile angezeigt wird, wo sie ebenfalls gestoppt werden kann.

### 🏷️ 2. Tags (Kategorien) erstellen
Aktivitäten können mithilfe von Tags in Kategorien gruppiert werden. Um Tags zu erstellen und zu konfigurieren, navigieren Sie vom „Heute“-Bildschirm zum Bereich „Verwalten“, indem Sie auf das entsprechende Symbol oben links klicken <img src="../../screenshots/scr_home_manage_button.png" height="200">, und wechseln Sie dann auf dem „Verwalten“-Bildschirm zum Tab „Tags“:

| Tag-Liste | Tag-Einstellungen |
|:---: | :---: |
| <img src="../../screenshots/scr_manage_tags.png" height="450"> | <img src="../../screenshots/scr_home_add_tagpicker.png" height="450"> |

Die Schaltfläche „+“ oben rechts auf diesem Tab dient zum Erstellen eines Tags (ähnlich wie beim Erstellen einer Aktivität). Klicken Sie auf einen Tag in der Liste (linkes Bild), um ihn zu konfigurieren (rechtes Bild).

### 📊 3. Visuelle Berichte anzeigen
Wechseln Sie zum Bildschirm „Bericht“, indem Sie auf das Symbol in der unteren Leiste klicken. Ein Kreisdiagramm zeigt Ihnen das Gesamtbild Ihrer Aktivitäten für den Tag, die Woche, den Monat oder das Jahr. Sie können Ihre Produktivität nach Aktivitäten oder Tags analysieren, indem Sie den Diagrammmodus über das Symbol oben rechts umschalten. Die Pfeile über diesem Symbol wechseln den Tag für den Bericht.

| Bericht nach Aktivitäten | Bericht nach Tags |
|:---: | :---: |
| <img src="../../screenshots/scr_report_activities.png" height="450"> | <img src="../../screenshots/scr_report_tags.png" height="450"> |

### ⚡ 4. Schnellzugriff konfigurieren (Schnellbedienfeld)
Sie können die Zeiterfassung direkt aus der Benachrichtigungsleiste verwalten. Fügen Sie dazu Aktivitäten zum Schnellbedienfeld hinzu — so können Sie das Tracking starten/stoppen, ohne die App zu öffnen oder das Gerät zu entsperren. Halten Sie dazu eine Aktivität auf dem „Heute“-Bildschirm lange gedrückt oder aktivieren Sie die entsprechende Option im Einstellungsfenster der Aktivität unter „Verwalten“ (siehe Punkt 1):

| langes Drücken auf dem „Heute“-Bildschirm | aus dem Fenster „Verwalten“ |
|:---: | :---: |
|<img src="../../screenshots/scr_notify_panel_set_add.png" height="450"> <img src="../../screenshots/scr_notify_panel_set_remove.png" height="450"> | <img src="../../screenshots/scr_home_add_toggle-panel.png" height="450"> |

Aktivitäten in der Benachrichtigungsleiste sehen wie folgt aus:

| in der Benachrichtigungsleiste | auf dem Sperrbildschirm |
|:---: | :---: |
|<img src="../../screenshots/scr_notification.png" height="450"> | <img src="../../screenshots/scr_notification_in_block.png" height="450"> |
